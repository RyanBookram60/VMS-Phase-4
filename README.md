This project holds all of the phase 4 implementations for the Data Managment System project, including a GUI interface for the user to interact with and a SQLite database connection.
This system implements the ability to store vehicle attributes and manipulate them into a SQLite database using the GUI.
______________
The system allows users to:
- Add single vehicles using a specified format
- Add vehicles through a text file using a file path
- Remove vehicles from the system using an interactive list
- Updates vehicles using a matching VIN number
- Estimate a vehicle's condition
